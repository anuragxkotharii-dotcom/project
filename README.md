# SCP Foundation | Secure Database  

A fictional **SCP Foundation Secure Database** built with HTML and CSS.  
This project simulates a classified archive of SCP objects, complete with navigation, containment procedures, and descriptions.  

---

## 📂 Project Structure  

project/
│── index.html          # Homepage with SCP overview and navigation
│── SCP-002.html        # SCP-002 containment procedures and description
│── SCP-003.html        # SCP-003 containment procedures and description
│── SCP-004.html        # SCP-004 containment procedures and description
│── SCP-005.html        # SCP-005 containment procedures and description
│── SCP-006.html        # SCP-006 containment procedures and description
│── css/
│    └── style1.css     # Stylesheet for layout and design

---

## 🚀 Features  

- **Navigation Bar**: Quick access to SCP entries (002–006).  
- **Homepage**: Warning messages, SCP cards, and links to individual SCP files.  
- **Individual SCP Pages**: Each SCP file contains:  
  - Item number and object class  
  - Special containment procedures  
  - Description of the SCP object  
  - Additional notes and references  
- **Responsive Design**: Uses `meta viewport` for mobile compatibility.  
- **Immersive Visuals**: Integrated SCP-related images for storytelling.
